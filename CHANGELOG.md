# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-12
### Added
- Initial release of Monopoly Ant.
- Support for 2-4 players (Pass & Play).
- Dynamic board generation (Medium size).
- Property purchasing and upgrading mechanics.
- Chance card system.
- Tax and Jail mechanics.
- Localization support (English and Vietnamese).
- Local persistence using Hive (Auto-save/Resume).
- Core UI components (Board, Tiles, Player Panels, Game Log).
- Technical documentation (PRD and Technical Overview).
- MIT License and Contributing guidelines.
